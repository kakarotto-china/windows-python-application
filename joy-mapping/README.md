# 配置参数说明
## 启动参数
```text

使用管理员启动

可以使用的启动参数有: -interval 或者 -i 后面接数字, 来设置模拟键盘按下的时间间隔, 默认不设置时为100, 例如:

打开cmd执行

./joy-mapping.exe -interval 500
按键时间间隔为500ms

./joy-mapping.exe
按键时间间隔为100ms
```
